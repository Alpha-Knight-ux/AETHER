# Contributing

Thank you for your interest in AETHER!

Although AETHER is currently closed-source, community feedback is greatly appreciated.

You can contribute by:

- Reporting bugs
- Suggesting new features
- Sharing ideas
- Improving documentation
- Testing new releases

## Bug Reports

Please include:

- A clear description
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (if applicable)
- Windows version

## Feature Requests

Please explain:

- What feature you'd like
- Why it would be useful
- Possible implementation ideas

Thank you for helping improve AETHER!
