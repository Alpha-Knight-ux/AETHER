---

## Reporting a Vulnerability

If you discover a security vulnerability in AETHER, please do not disclose it publicly.

Instead, create a private GitHub issue or contact the developer directly.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Screenshots (if applicable)
- System information
- Affected version

Security reports will be investigated as quickly as possible.

Thank you for helping keep AETHER secure.
